# 时间序列预测 (Time Series Prediction)

本部分记录了时间序列预测方向的学习笔记，重点关注深度学习方法在金融时间序列中的应用。

时间序列预测的核心挑战在于：如何从历史数据中提取有效的时域和频域特征，以提高对未来走势的预测精度。笔记围绕以下几条技术路线展开：

- **LSTM + 频率分解**：基于 SFM（State-Frequency Memory）模型，将 LSTM 的状态向量扩展为状态-频率矩阵，使模型能同时捕获时域和频域特征，适用于不同投资周期的预测需求。
- **小波变换 + 注意力机制**：利用小波变换将时间序列分解为多尺度成分，结合注意力机制融合局部时频特征和全局趋势。
- **多变量 LSTM + 注意力**：探索如何将多个输入变量（如多只股票、多个因子）纳入 LSTM 框架，并通过注意力机制自动学习变量间的重要性权重。
- **小波变换 + 投资组合构建**：WISE 方法利用小波分析编码股票嵌入向量，从时域和频域两个维度估计股票相关性，用于风险厌恶型投资组合的构建。
- **SDF 与回报空间**：从资产定价理论的角度理解时间序列预测问题。
- **特征视角与数据预处理**：讨论不同特征构建方式对预测效果的影响。

这些笔记既包含模型原理的推导，也记录了实验结果和个人思考。
