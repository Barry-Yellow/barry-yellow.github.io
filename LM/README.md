# 语言模型 (Language Model)

本部分整理了自然语言处理和大语言模型相关的基础知识笔记，从经典 NLP 方法到现代大模型技术。

笔记覆盖以下内容：

- **NLP 基础知识**：词嵌入（Word2Vec、GloVe）、上下文相关表示（ELMo）等经典 NLP 技术，是理解后续模型的基础。
- **Transformer**：详解 Transformer 架构的核心组件——多头自注意力、位置编码、前馈网络，以及编码器-解码器结构。
- **BERT**：基于 Transformer 编码器的预训练语言模型，通过掩码语言模型（MLM）和下一句预测（NSP）两个预训练任务学习双向语言表示。
- **大模型基础**：Prompt Engineering、RAG（检索增强生成）、微调等大模型应用技术的概述。
- **强化学习基础**：REINFORCE、Actor-Critic、PPO 等强化学习算法，这些是理解 RLHF（基于人类反馈的强化学习）的前置知识。
- **MoE (Mixture of Experts)**：混合专家模型的门控机制和稀疏激活策略，是大模型扩展的重要技术路线。

这些笔记从基础到前沿，构成了理解现代大语言模型的知识体系。
