# Mamba 架构

本部分记录了 Mamba 模型及其相关实验的学习笔记。Mamba 是一种基于状态空间模型（SSM）的序列建模架构，旨在解决 Transformer 注意力机制随序列长度二次增长的计算瓶颈。

笔记围绕以下内容展开：

- **Mamba 理论基础**：从 SSM 的基本方程出发，介绍离散化、递归表示与卷积表示的等价性，然后讲解 S4 模型如何通过 HiPPO 矩阵实现长序列建模，最后引出 Mamba 的核心创新——选择性状态空间（Selective SSM），即让 A、B、C 矩阵依赖于输入，实现类似 RNN 门控的内容感知能力。
- **Mamba + 频率分解实验**：将 Mamba 与频率分解方法结合，探索其在不同频率成分上的建模能力。
- **Mamba 改进**：记录了对 Mamba 架构的改进实验和结果分析。

Mamba 的核心优势在于：训练时可以像 CNN 一样并行计算，推理时可以像 RNN 一样顺序执行，且计算量与序列长度呈线性关系。
