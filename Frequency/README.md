# 频域分析 (Frequency Analysis)

本部分整理了频域分析的基础理论知识，为后续频域 Transformer 等模型的学习提供数学基础。

频域分析的核心思想是：将时域信号通过傅里叶变换、小波变换等工具转换到频域，从而揭示信号中隐藏的周期性成分和多尺度结构。笔记包含以下内容：

- **自相关系数、自谱密度与互协方差**：从自相关函数出发，推导自谱密度函数（功率谱），并扩展到多变量情形下的互协方差和共谱密度。这些是频域分析最基本的数学工具。
- **时序数据平稳方法**：对比 EMD（经验模态分解）、X11 季节调整、差分等常用的时间序列平稳化方法，讨论各自的适用场景和优缺点。
- **SpectFormer 频率注意力**：SpectFormer 提出将频率域的谱注意力与标准自注意力层次化结合，在计算机视觉任务中取得了优异效果。（详见频域 Transformer 部分）

这些基础知识是理解 FEDformer、AutoFormer 等频域增强模型的前置条件。
