## 🚀 快速开始

### 在线访问
直接访问 [GitHub Pages](https://barry-yellow.github.io) 即可浏览所有笔记。

### 本地运行
```bash
# 克隆仓库
git clone https://github.com/Barry-Yellow/barry-yellow.github.io.git

# 进入目录
cd barry-yellow.github.io

# 使用Python启动本地服务器
python -m http.server 3000

# 或使用Node.js
npx docsify-cli serve
```

然后在浏览器中打开 `http://localhost:3000`
