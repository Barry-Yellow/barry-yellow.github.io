# 深度多模态学习 (Deep Multimodal Learning)

本部分记录了多模态学习领域的核心模型和技术笔记，重点关注视觉-语言多模态模型的发展脉络。

多模态学习的核心问题是：如何让模型理解和关联来自不同模态（如图像、文本、视频）的信息？笔记按照技术演进路线组织：

- **多模态学习基础**：介绍多模态学习的基本概念、主流方法分类（对比学习、生成式、融合式），以及 SimCLR、MOCO 等自监督学习方法。
- **CLIP**：OpenAI 提出的对比学习框架，通过 4 亿图文对训练实现文本与图像的语义对齐，开创了零样本跨模态理解的范式。
- **BLIP**：统一了多模态理解和生成任务，提出 CapFilt 机制利用噪声网络数据进行高效预训练。
- **BLIP-2**：通过 Q-Former 桥接冻结的视觉编码器和语言模型，大幅降低了多模态预训练的计算成本。
- **Diffusion 模型**：扩散模型的数学原理（前向加噪、反向去噪、DDPM、DDIM），是当前图像生成的主流方法。
- **自回归 vs 扩散**：对比自回归生成和扩散生成两种范式的优劣。
- **NLP 学习笔记**：NLP 基础知识在多模态场景中的应用。
- **VidKV**：视频大语言模型中 KV Cache 的量化压缩方法，从多模态视角分析。
