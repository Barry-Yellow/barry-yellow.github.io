# 多模态概念笔记

> 本文记录了多模态学习中的核心概念和零散知识点，作为学习过程中的参考。

## 1. 模态数据对齐 (Modality Alignment)

模态数据对齐是指将不同模态的数据在语义、时间或空间上建立对应关系。

### 1.1 语义对齐
将不同模态数据的语义信息映射到同一特征空间。例如 CLIP 将图像和文本映射到共享的 768 维向量空间，通过余弦相似度衡量语义匹配程度。

### 1.2 时间对齐
在时序数据（视频、音频）中建立模态间的时间同步关系。例如视频理解任务中，将语音与画面动作在时间轴上精确同步。

### 1.3 空间对齐
在空间维度上对齐不同模态数据。例如图像分割任务中，将文本指令"红色汽车"与图像中的对应区域对齐。

## 2. Shot 学习范式

| 类型 | 定义 | 典型应用 |
|------|------|----------|
| Zero-Shot | 无目标任务样本，靠预训练知识迁移 | CLIP 零样本图像分类 |
| One-Shot | 仅用单个样本学习新任务 | LLaVA 单图视觉问答 |
| Few-Shot | 少量样本（5-20 个）微调或提示学习 | Flamingo 跨语言图文对齐 |
| In-Context Learning | 在输入中嵌入示例，模型隐式学习 | GPT-4V 多模态推理 |

## 3. 多模态模型的训练范式

多模态模型通常分两个阶段训练：
1. **预训练阶段**：冻结图像编码器和语言模型，只训练连接两者的 projector（投影层）
2. **指令微调阶段**：解冻部分或全部参数，用指令数据进行微调。跨注意力层在此阶段处于激活状态

### 多模态融合方式
- **统一 LM 框架**：将不同模态数据统一编码后放入同一个模型（如 Chameleon）
- **跨模态注意力**：将图像编码放入多头注意力层的 Key/Value，文本作为 Query（如 Flamingo、BLIP-2）

## 4. 评估指标

- **FID**：生成图像与真实图像在 Inception 特征空间中的分布距离。需要采样大量图片估算高斯分布参数。公式：$\text{FID} = \|\mu_1 - \mu_2\|^2 + \text{Tr}(\Sigma_1 + \Sigma_2 - 2\sqrt{\Sigma_1 \Sigma_2})$
- **CLIP Score**：用 CLIP 模型计算生成图像与文本描述的余弦相似度

## 5. 生成模型笔记

- 自回归模型可以减少生成中未提及内容的出现
- 扩散模型倾向于生成新内容，而非修改现有内容
- Diffusion + 自回归的结合可以兼顾生成质量和可控性
- VQVAE 的 VQ tokenizer 训练难度较大
- 一次性生成多图可以提升风格一致性
